CoreAI

프로젝트 개요

- GPT-2 계열의 casual language model 을  
- 자체 토크나이저(CoreAI_Tokenizer)와 함께  
- 대규모 JSONL 텍스트 데이터셋을 sliding-window 방식으로 전처리하여  
- Mixed-Precision, Gradient Accumulation, Adafactor 옵티마이저, Cosine LR 스케줄러를 적용해  
- 사전학습(pre-training)을 수행을 수행하며
- 해당 프로젝트로 LLM 기본 모델 구축에 대한 이해 및 경험을 쌓는 것을 목표로 진행햐였습니다.
